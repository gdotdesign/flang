# Colors
Colors can be represented in many formats:
  * **rgb** - rgb(0,0,0)
  * **rgba** - rgba(0,0,0,1)
  * **hsl** - hsl(0,100%,100%)
  * **hsla** - hsl(0,100%,100%,1)
  * **hex** - #000000
  * **keyword** - black
